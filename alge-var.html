What is a variable?

A variable is a place holder in an equation or expression that represents something whose value is unknown. Often, these placeholders are represented by letters from the alphabet, the most common being x and y, but because the only aim is to make a symbol to represen the unknown you can use all kinds if symbols like triangle ▲ . Becuase we’re trying to find the value of our unknown, you can treat algebra as a game where you use inverse operations on both sides to balence the equation into a simpler form where the variable is isolated
