Circumference

Circumference is a special term used to describe the perimeter of a circle. In order to calculate it’s value, you need to have the value of the diameter of the circle. If that is unknown while the radius is, you can find the value of the diameter by multiplying the radius by 2. The formula for the circumference of a circle is, c = dπ
