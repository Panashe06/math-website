Order of operations

Up until this moment, youâ€™ve probably been used to solving equations from left to right. This method does not work when attempting to solve equations with different types of operators. When we experiance this we follow the agreed upon order of operations to correctly interpract an equation. The order of operations is:

  1. Parenthesis
  2. Exponents
  3. Multiplication / Division (both share equal importance in step three)
  4. Addition / Subtraction

If both multiplication and division are in the same equation, then we interpret whatever operation comes first in the equation when read from left to right, the same applies for addition and subtraction.
